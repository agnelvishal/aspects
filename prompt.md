Need to find conjunctions between 2 or more planets across a time range.

It can be frontend only web app. Use swisseph-wasm whose readme is in swisseph-wasm/README.md 

You can also read other files related to swisseph-wasm in swisseph-wasm directory.

Sun (Surya), moon (Chandra), Mars (Mangala), Mercury (Budha), Jupiter (Brihaspati), Venus (Shukra), Saturn (Shani), Rahu and ketu are the planets to be considered.
